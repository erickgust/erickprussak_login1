## Usage

Clone this project:

```sh
git clone git@github.com:erickgust/esther.git
```

Install the dependencies:

```sh
yarn
```

Run mobile version

```sh
yarn start
```

Run web version (use Node 16 LTS for this)

```sh
yarn web
```

## Production Website

To generate the production version, you can run (use Node 16 LTS):

```sh
npx expo export:web
npx serve web-build
```
